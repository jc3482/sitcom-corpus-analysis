INFORMATION RETRIEVAL PACKAGE - Reference
===============================================

STRUCTURE:
----------
information_retrieval/
├── __init__.py      # Package initialization
├── __main__.py      # CLI entry point
└── search.py        # Core search functions

USAGE:
------

1. Search one show:
   python -m information_retrieval search friends "coffee shop"
   python -m information_retrieval search tbbt "physics&quantum" --top 10

2. Search all shows:
   python -m information_retrieval search-all "wedding"
   python -m information_retrieval search-all "proposal&ring" --top 5

3. List available shows:
   python -m information_retrieval list

SHOWS:
------
friends, tbbt, seinfeld, office, modern_family

QUERY SYNTAX:
-------------
"coffee shop"    - Exact phrase
"coffee&shop"    - AND (both must appear)
"coffee/shop"    - OR (either must appear)

IMPORT AS LIBRARY:
------------------
from information_retrieval import search_episodes, load_show_data

episodes = load_show_data('friends')
results = search_episodes(episodes, "coffee&shop", top_k=5)
